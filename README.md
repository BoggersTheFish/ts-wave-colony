# TS-WaveColony

**Multi-Agent Wave Colony for the Thinking System**

A native multi-agent coordination layer where independent Tension Graph agents share a common wave field, interfere constructively/destructively, and resolve *collective tensions*.

This is the beginning of true emergent collective intelligence within the TS architecture.

## Quick Start

```bash
git clone https://github.com/BoggersTheFish/ts-wave-colony.git
cd ts-wave-colony
pip install -r requirements.txt
python -m ts_colony.run_example
```

## Philosophy

In single-agent TS, one graph propagates waves and resolves its own tensions.

In **WaveColony**, multiple agents exist as sub-graphs. They:
- Share activation waves
- Create interference patterns
- Form collective attractors
- Resolve higher-order tensions together

This mirrors real biological colonies, neural ensembles, and societies.

## Core Concepts

- **Agent**: Independent TensionGraph with its own goals/constraints
- **Shared Field**: Common wave propagation medium
- **Interference**: Constructive / destructive wave interaction between agents
- **Collective Tension**: Emergent constraints that no single agent can resolve alone
- **Consensus via Relaxation**: The colony relaxes into coherent states
